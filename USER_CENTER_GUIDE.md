## 🚀 快速开始

### 1. 启动项目

```bash
# 安装依赖（如果尚未安装）
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:5173/ 或 http://localhost:5176/（取决于端口占用情况）

### 2. 登录管理员账号

- **邮箱**: admin@example.com
- **密码**: admin123

### 3. 进入用户中心

点击导航栏右上角的用户头像，选择"用户中心"进入管理后台。

## 📋 功能详解

### 1. 仪表盘 (Dashboard)

#### 管理员视图
- **统计卡片**: 显示总用户数、固件总数、下载次数、捐赠总额
- **待审核固件**: 列出所有待审核的固件，可快速通过或拒绝
- **快捷操作**: 一键进入各个管理模块

#### 普通用户视图
- **下载统计**: 查看个人下载情况
- **账户概览**: 基本信息展示

### 2. 分类管理 (管理员)

类似 WordPress 的分类系统：

```
📁 慧荣 (SMI)
  ├── SM2258XT
  ├── SM2259XT
  └── SM2263XT
  
📁 群联 (Phison)
  ├── PS3111
  └── PS5013
  
📁 联芸 (Maxio)
  └── MAP1202
```

**功能**:
- ✅ 添加新分类
- ✅ 编辑分类名称
- ✅ 设置分类顺序
- ✅ 删除分类（无关联固件时）
- ✅ 支持多级分类（父分类 → 子分类）

### 3. 用户管理 (管理员)

用户列表管理，包括：

- **用户信息**: 昵称、邮箱、注册时间
- **角色管理**: 
  - 👑 管理员 (admin): 完全访问权限
  - 🛡️ 维护者 (maintainer): 固件管理权限
  - 👤 普通用户 (user): 基础访问权限
- **会员状态**: Premium / 普通
- **下载统计**: 用户下载次数

**操作**:
- 修改用户角色
- 删除用户（不能删除自己）
- 查看用户详情

### 4. 固件管理 (维护者/管理员)

固件的完整生命周期管理：

#### 状态流转
```
上传 → 待审核(pending) → 已通过(approved) / 已拒绝(rejected)
```

#### 功能列表
- **上传固件**: 添加新的开卡工具
- **审核固件**: 通过或拒绝用户上传的固件
- **编辑固件**: 修改固件信息
- **删除固件**: 移除不需要的固件
- **查看统计**: 下载次数、状态等

#### 固件信息
- 标题和描述
- 版本号
- 所属分类
- 是否付费及价格
- 上传者信息
- 下载次数

### 5. 网站设置 (管理员) - ⭐ 核心功能

类似 WordPress 的自定义设置：

#### 5.1 基本设置
- **网站名称**: "SSD开卡工具站"
- **网站描述**: 副标题/简介

#### 5.2 模块开关
控制首页显示哪些模块：

| 模块 | 说明 |
|------|------|
| 🦸 英雄区域 | 首页顶部大幅宣传区 |
| 🔥 热门工具 | 热门固件展示 |
| ✨ 最新上传 | 最新固件列表 |
| 💰 捐赠公示 | 捐赠记录展示 |
| 🏆 贡献榜单 | 贡献者排行榜 |

#### 5.3 下载配额
设置用户下载限制：

- **免费用户**: 每月 5 次下载
- **Premium 用户**: 每月 100 次下载

#### 5.4 主题配色 ⭐ 实时预览
自定义网站颜色主题：

- **主色调**: 网站主按钮、重要强调色
- **辅助色调**: 次要元素、hover 状态

**实时预览**: 修改颜色后立即看到效果展示

## 🔧 技术实现

### 前端技术栈
- React 18 + TypeScript
- TailwindCSS 样式
- Framer Motion 动画
- Lucide React 图标
- React Router 路由
- Zustand 状态管理

### 后端 API
- Express + TypeScript
- MySQL 数据库
- RESTful API 设计
- JWT 认证（可选）

### 数据库表
- **users**: 用户表
- **categories**: 分类表（树形结构）
- **firmware**: 固件表
- **downloads**: 下载记录表
- **donations**: 捐赠记录表
- **config**: 系统配置表

## 📝 日常使用流程

### 管理员日常工作

1. **登录后台**: 使用管理员账号登录
2. **查看仪表盘**: 了解网站整体状况
3. **审核固件**: 处理用户上传的新固件
4. **管理分类**: 根据需要添加/调整分类
5. **调整设置**: 根据运营需要修改网站配置
6. **用户管理**: 管理用户角色和权限

### 维护者日常工作

1. **登录后台**: 使用维护者账号登录
2. **上传固件**: 添加新的开卡工具
3. **审核固件**: 处理其他用户上传的固件
4. **查看统计**: 了解固件下载情况

### 普通用户

1. **浏览固件**: 查看和搜索固件
2. **下载固件**: 使用每月的下载额度
3. **管理账户**: 查看下载记录和个人信息

## 🚨 注意事项

1. **权限安全**: 只有管理员可以访问分类管理、用户管理和网站设置
2. **数据备份**: 定期备份数据库，防止数据丢失
3. **密码安全**: 生产环境请修改默认管理员密码
4. **文件上传**: 固件上传功能需要配置服务器文件存储
5. **实时预览**: 主题配色修改仅在设置页面实时预览
