# SSD 开卡工具站 - WordPress 风格后台管理系统

## 用户中心完整功能设计

### 核心概念
- 统一的用户中心入口，根据角色显示不同功能
- 类似 WordPress 后台的管理界面
- 左侧固定导航 + 右侧内容区域

### 用户角色权限

| 功能模块 | 管理员 | 维护者 | 普通用户 |
|---------|--------|--------|----------|
| 仪表盘统计 | ✅ 完整 | ✅ 简化 | ✅ 简化 |
| 账户信息 | ✅ | ✅ | ✅ |
| 下载记录 | ✅ | ✅ | ✅ |
| 我的固件 | ✅ | ✅ | ❌ |
| **分类管理** | ✅ | ❌ | ❌ |
| **用户管理** | ✅ | ❌ | ❌ |
| **固件审核** | ✅ | ✅ | ❌ |
| **网站设置** | ✅ | ❌ | ❌ |

### 详细功能

#### 1. 仪表盘
- 统计卡片：用户总数、固件总数、下载次数、捐赠总额
- 待审核固件数量（维护者/管理员）
- 快捷操作入口

#### 2. 账户信息
- 个人信息展示和编辑
- 修改密码
- 角色标识

#### 3. 下载记录
- 下载历史列表
- 下载额度显示

#### 4. 分类管理（管理员）
- 树形分类列表
- 添加/编辑/删除分类
- 拖拽排序

#### 5. 用户管理（管理员）
- 用户列表（分页、搜索）
- 修改用户角色
- 删除用户

#### 6. 固件管理（维护者/管理员）
- 固件列表（所有状态）
- 上传新固件
- 审核固件（通过/拒绝）
- 编辑/删除固件

#### 7. 网站设置（管理员）
- **网站基本设置**：名称、描述、Logo
- **模块开关**：首页显示哪些模块
- **下载配额**：免费/Premium 额度设置
- **主题配色**：自定义网站配色
- **实时预览**：修改后立即预览效果

### 技术实现
- React + TypeScript
- TailwindCSS 样式
- 组件化开发
- API 调用封装
