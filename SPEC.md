# SSD 开卡工具站 - 用户中心管理系统

## 1. Concept & Vision

一个类似 WordPress 的后台管理系统，所有自定义功能都集成在用户中心内。根据用户角色（管理员、维护者、普通用户）显示不同的管理选项。管理员可以管理整个网站，包括分类、用户、固件和系统配置。

## 2. Design Language

- **美学方向**: WordPress 风格的管理后台，左侧固定导航 + 右侧内容区域
- **配色方案**: 
  - 主色: `#1e40af` (深蓝)
  - 次色: `#3b82f6` (亮蓝)
  - 强调色: `#10b981` (绿色)
  - 背景: `#f3f4f6` (浅灰)
  - 文字: `#111827` (深灰)
- **字体**: Inter, system-ui, sans-serif
- **布局**: 左侧固定导航栏 (240px) + 右侧内容区域
- **动效**: 平滑的页面切换和状态变化

## 3. Layout & Structure

### 左侧导航栏
- Logo 和网站名称
- 用户头像和角色标识
- 导航菜单项（根据角色显示不同选项）
- 登出按钮

### 右侧内容区
- 顶部面包屑导航
- 内容卡片
- 操作按钮和表单

## 4. Features & Interactions

### 用户角色权限

| 角色 | 仪表盘 | 账户信息 | 下载记录 | 我的固件 | 分类管理 | 用户管理 | 固件审核 | 网站设置 |
|------|--------|---------|---------|---------|---------|---------|---------|---------|
| 管理员 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| 维护者 | ✅ | ✅ | ✅ | ✅ | ❌ | ❌ | ✅ | ❌ |
| 普通用户 | ✅ | ✅ | ✅ | ❌ | ❌ | ❌ | ❌ | ❌ |

### 核心功能

#### 1. 仪表盘 (Dashboard)
- 显示统计数据卡片（用户数、固件数、下载次数、捐赠总额）
- 待审核固件提示
- 快捷操作入口

#### 2. 账户信息
- 显示当前用户信息
- 编辑昵称、邮箱
- 修改密码

#### 3. 下载记录
- 显示用户下载历史
- 显示剩余下载额度

#### 4. 分类管理 (管理员)
- 树形分类列表
- 添加/编辑/删除分类
- 设置分类顺序

#### 5. 用户管理 (管理员)
- 用户列表
- 搜索和筛选
- 修改用户角色
- 删除用户

#### 6. 固件管理 (维护者/管理员)
- 固件列表（所有状态）
- 上传新固件
- 审核固件（通过/拒绝）
- 编辑/删除固件

#### 7. 网站设置 (管理员)
- 网站基本设置（名称、描述、Logo）
- 模块开关（首页显示哪些模块）
- 下载配额设置（免费/Premium）
- 主题配色配置
- **实时预览功能**

## 5. Component Inventory

### 导航菜单项
- 默认: 灰色文字，白色背景
- 悬停: 蓝色文字，浅蓝色背景
- 激活: 蓝色文字，左侧蓝色边框

### 统计卡片
- 白色背景，圆角
- 图标 + 数字 + 标签
- 悬停: 轻微上浮效果

### 数据表格
- 表头: 深灰色背景
- 行: 白色背景
- 悬停: 浅灰色背景
- 操作按钮: 小型图标按钮

### 表单组件
- 输入框: 白色背景，灰色边框
- 聚焦: 蓝色边框
- 标签在输入框上方

### 模态框
- 半透明黑色背景
- 居中白色卡片
- 动画: 淡入 + 缩放

## 6. Technical Approach

### 前端
- React 18 + TypeScript
- React Router v6
- Zustand 状态管理
- TailwindCSS 样式
- Lucide React 图标
- Framer Motion 动画

### 后端 API
- Express + TypeScript
- MySQL 数据库
- RESTful API 设计

### 数据模型
- Users: 用户表
- Categories: 分类表
- Firmware: 固件表
- Downloads: 下载记录表
- Donations: 捐赠记录表
- Config: 系统配置表
